# TC39 Stage 3 Decorator + Babel + Webpack Example

## Environments

- Node.js 20.12.2
- React 16.14.0
- Webpack 5.91.0
- Babel 7.24.4

## How to

- Build.
  ```
  $ npm run build
  ```
- Serve.
  ```
  $ npx serve ./dist
  ```
- Enter `http://localhost:3000/` in your browser.
- Check the console.
  ```
  Uncaught ReferenceError: Abc is not defined
  ```
